# HR-Analytics-Dashboard-PowerBI
A Power BI project focused on HR analytics, including employee attrition analysis, workforce trends, and data-driven insights.


## Project Overview
This project analyzes employee data using Power BI to understand attrition,
employee demographics and workforce trends.

## Tools Used
- Power BI
- Excel

## Dashboard Pages
1. Attrition Analysis
2. Workforce Analysis

## Key Insights
- Total employees analyzed
- Attrition rate
- Department wise attrition
- Age group analysis
- Employee distribution
